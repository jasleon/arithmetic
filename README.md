# arithmetic
![build](https://github.com/jasleon/arithmetic/actions/workflows/main.yml/badge.svg)

Experiment with GitHub actions in a C++ project

## Dependencies
- cmake
- make
- gcc/g++

## Build Instructions
1. Clone this repo.
2. Make a build directory: `mkdir build && cd build`
3. Compile: `cmake .. && make`
4. Run it: `./app`
